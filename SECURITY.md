# Security Policy

Report vulnerabilities to disapproval@derails.dev. Do not open a GitHub issue.

## Out of Scope

- CSRF, DNSSEC, CAA, CSP headers
- DNS or email security related
- Rate Limiting
- Temporal paradoxes caused by interns (see INCIDENT_REPORTS.md)
